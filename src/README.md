# pcd2image
这段算法中主要实现的是根据给定的四个标定参数将点云均匀渲染到图像上，为了保证输出的投影图效果，可以调整点的大小，是否伪深度着色，最大距离和降采样比例等。
该算法支持ascii和二进制pcd文件，并分别将相机的内参矩阵，畸变系数和外参矩阵（旋转/平移）存放到三个不同的txt文件中，方便计算时调用。
内参矩阵：intrinsics.txt
畸变系数：dist_coeffs.txt
外参矩阵：extrinsics.txt

# pcd2image_lcc
该算法从上面的原始代码出发，优化了部分参数，中间过程和渲染效果，使其更接近livox_camera_calib效果
算法利用了体素降采样，使得效果不易受不均匀波状点云影响，但该体素计算在科学严谨性上略有不足，算法在渲染时还利用特征提取给原图像增加了边缘显示的能力，更方便观察投影效果中的边缘是否和不同深度的点云对齐。
该算法支持ascii和二进制pcd文件，并分别将相机的内参矩阵，畸变系数和外参矩阵（旋转/平移）存放到三个不同的txt文件中，方便计算时调用。
内参矩阵：intrinsics.txt
畸变系数：dist_coeffs.txt
外参矩阵：extrinsics.txt
